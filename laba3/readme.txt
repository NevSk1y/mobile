1. Необходимо скачать папку num2t4ru по ссылке https://github.com/seriyps/ru_number_to_text и вставить в папку с файлом программы
2. Скайчайте файлы output.csv, data.csv, schetinput.docx (данный файл скачать в одну папку с файлом программы)
3. Запустите Miloserdov_laba3.py при помощи python 3.7.7 и любой IDE (прим. Wing 101 7.2)
4. Введите путь к файлам output.csv и data.csv
5. Укажите необходимый IP-адрес и номер телефона, для которых необходимо провести тарификацию
6. Результат будет выведен в файл oplata.pdf


Содержание папки:
Miloserdov_laba3.py - файл с исходным кодом
laba3 Miloserdov.pdf - отчет по работе
output.csv - файл с трафиком NetFlow
data.csv - файл с исходными данными тарификации
oplata.pdf - файл с итоговым счетом
schetinput.docx - файл-шаблон для вставки данных
