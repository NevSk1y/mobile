1. Запустите Miloserdov_laba1.py при помощи python 3.7.7 и любой IDE (прим. Wing 101 7.2)
2. Введите путь к файлу CDR и его имя (например, D:\data.csv)
3. Введите телефонный номер, для которго необходимо произвести тарификацию
4. Результат будет выведен в командную строку

Содержание папки:
Miloserdov_laba1.py - файл с иходным кодом
data.csv - файл с исходными данными
