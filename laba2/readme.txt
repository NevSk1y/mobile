1. Необходимо скачать библиотеку matplotlib через pip
2. Скайчайте файл output.csv
3. Запустите Miloserdov_laba2.py при помощи python 3.7.7 и любой IDE (прим. Wing 101 7.2)
4. Введите путь к файлу output.csv
5. Укажите необходимый IP-адрес, для которого необходимо провести тарификацию (по варианту он 217.15.20.194)
6. Результат будет выведен в командную строку
7. В открышемся диалоговом окне будет представлен график зависимости трафика от времени

Содержание папки:
Miloserdov_laba2.py - файл с исходным кодом
laba2 Miloserdov.pdf - отчет по работе
output.csv - файл с трафиком NetFlow
plot.png - скриншот графика
result2.png - скриншот результата
